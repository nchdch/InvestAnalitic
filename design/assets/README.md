# Assets

Original, minimal brand assets created for InvestAnalitic. There was **no prior
logo or asset library** in the codebase (`src/assets` was empty), so these are
**placeholder identity** — easy to replace if the team commissions a real logo.

| File | Use |
|---|---|
| `logo.svg` | Full horizontal lockup: spark mark + "InvestAnalitic" wordmark. |
| `logo-mark.svg` | Standalone mark (favicon, avatar, collapsed sidebar). |

## The mark
An ascending analytic line with an up-and-right arrowhead and three tracked data
points — "track + analyze + grow." Azure tile (`#2C5FD4`, the brand `--accent`),
white line, light-azure nodes. Geometric, flat, no gradient.

## Wordmark
"Invest" in ink, "Analitic" in azure — Golos Text Bold, −0.4 tracking. The split
reinforces the analysis half of the product.

## Broker logos
Real broker brands (Сбер, Т-Банк, ВТБ, …) are **not** reproduced here — they are
third-party trademarks. The UI kit represents accounts with neutral initial-tiles
(`BrokerTile` pattern) using brand-neutral fills. Drop in licensed broker marks
if/when permitted.

## Iconography
UI icons come from **Lucide** (CDN) — see `readme.md` § Iconography. Not stored here.
