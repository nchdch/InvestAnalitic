# Handoff: InvestAnalitic Design System → код

## Overview
Это **полная дизайн-система** для InvestAnalitic — AI-first веб-сервиса учёта и
анализа инвестиционного портфеля (акции + облигации по нескольким счетам, P&L,
дивиденды/купоны, ребалансировка, ИИ-аналитик в центре продукта). Репозиторий
содержит токены, 14 компонентов, рабочий UI-кит всех экранов и гайдлайны.

> **Это и есть пакет хэндоффа.** Скачанный zip = весь этот проект. Пути ниже —
> от корня проекта. Цель: **перенести систему в репозиторий** `InvestAnalitic/`
> (React 18 + TypeScript + Vite + Zustand) — в пустые сейчас папки `src/theme`,
> `src/styles`, `src/components`. Дизайн делался под эту кодовую базу и доменные
> типы из `src/types/`.

## About the Design Files
Файлы — **дизайн-референс** в HTML/CSS + React-через-Babel (прототип). Это не
продакшн-код «скопировать как есть»: задача — **воссоздать компоненты и экраны в
среде репозитория** (Vite, TS, обычный `import`/`export`, CSS-модули или глобальный
CSS — как принято в проекте). Компоненты уже написаны как React-функции на токенах,
к каждому есть `.d.ts` с TypeScript-интерфейсом пропсов — переносить близко к 1:1.

## Fidelity
**High-fidelity.** Финальные цвета, типографика, отступы, состояния и взаимодействия.
Воспроизводить пиксель-в-пиксель. Все значения — в токенах; не хардкодить, тянуть из
CSS-переменных.

## Design Tokens (источник истины)
Все значения — CSS-переменные в `tokens/` (197 шт.). Ключевые ориентиры:

**Цвет (семантические роли — строить только на них):**
- Текст: `--text-1` (заголовки/цифры), `--text-2` (body), `--text-3` (вторичный), `--text-4` (placeholder)
- Поверхности: `--surface-app` (канва), `--surface-card` (белые карточки), `--surface-sunken` (инпуты/хедеры таблиц), `--surface-ai` (фон сообщений ИИ)
- Границы: `--border-1` (hairline), `--border-2` (инпуты), `--divider`
- Бренд: `--accent` (azure / cobalt), `--accent-hover`, `--accent-soft`
- **P&L (изолированы от бренда!):** `--pnl-up` (emerald), `--pnl-down` (red), `--pnl-flat`; мягкие — `*-soft`
- Статусы: `--positive --negative --warning --info`; ИИ — `--ai` (violet), `--ai-soft`
- Тёмная тема: `[data-theme="dark"]` переопределяет те же семантические токены — компоненты инвертируются автоматически.

**Типографика:** `--font-sans` (Golos Text), `--font-mono` (JetBrains Mono).
Шкала `--text-display/-h1…-h4/-lg/-base/-sm/-xs/-2xs`. Деньги и любые табличные
числа — `font-variant-numeric: tabular-nums lining-nums` (утилита `.ia-num` / `.ia-mono`).

**Отступы:** 4px-сетка `--space-1…15`. **Радиусы:** контролы `--radius-md` (8px),
карточки `--radius-lg` (12px), модалки `--radius-xl`. **Тени:** прохладные мягкие
`--shadow-xs…xl` + акцентное свечение `--shadow-accent`/`--shadow-ai` (только для
ИИ-композера). **Моушн:** `--dur-fast/base/slow` + `--ease-out/spring`.

## Компоненты (14)
У каждого в пакете лежит `.prompt.md` с примерами и `.d.ts` с полной сигнатурой —
читать при реализации. Кратко:

**forms/** — `Button` (primary/secondary/ghost/soft/danger · sm/md/lg · loading),
`IconButton`, `Input` (проп `numeric` → mono+tabular+right для денег/количеств),
`Select`, `Switch`.

**data/** — `Card` (hairline + мягкая тень; `tightBody` для таблиц),
`StatCard` (крупная цифра + дельта), **`PnLValue`** ⭐ (знак/цвет/стрелка/tabular,
авто-формат `+12 480,00 ₽ (+1,98%)`, истинный минус `−`), `Badge`,
`AllocationBar` (главный «график» — стек-полоса состава), `Avatar` (нейтральная
плитка брокера из инициалов).

**navigation/** — `Tabs` (line / pill).

**ai/** — **`AIMessage`** (реплика диалога: ИИ — градиентный «спарк» + фиолетовый
бабл + чипы-действия; пользователь — azure-бабл; `typing` для индикатора),
**`AIComposer`** ⭐ (постоянный инпут — позвоночник продукта; свободный ввод сделок
«Купил 5 лотов Сбера по 286» + вопросы; свечение акцента на фокусе).

## Экраны (эталон — `ui_kits/app/index.html`)

**App shell:** фикс-сайдбар 264px (лого, навигация, мини-список счетов, юзер) +
скролл-контент ≤1280px + топбар (поиск, уведомления, «Добавить сделку»).

1. **Портфель** — герой-итог (`2 480 350,00 ₽`) + дневной/общий P&L, `AllocationBar`
   состава, правый рейл «Аналитик заметил» (проактивные сигналы ⚠️/📅/🧾), таблица
   позиций с вкладками Акции/Облигации/Деньги (per-row P&L, вес, YTM, sticky-хедер).
2. **ИИ-аналитик** — лента `AIMessage` + закреплённый `AIComposer`; индикатор печати,
   чипы-подсказки.
3. **Ребалансировка** — полосы «текущий vs целевой вес» + сгенерированный план
   покупок/продаж + дисклеймер.
4. **Выплаты** — календарь дивидендов/купонов, форвардная доходность, статусы
   «выплачено / через N дней».

## Interactions & Behavior
- Tabs/Switch/Select — управляемые (`value`+`onChange`).
- `AIComposer.onSend(text)` — Enter отправляет, Shift+Enter перенос; textarea авто-рост.
- Ховеры быстрые (`--dur-fast`), без сдвига лэйаута; фокус — кольцо `--focus-ring` 3px.
- Уважать `prefers-reduced-motion` (в `tokens/base.css` уже есть).

## Content & House Rules (обязательно)
- Язык интерфейса — русский, обращение на **«ты»**; тон — конкретный аналитик с
  цифрами из портфеля пользователя.
- Числа: `1 234 567,89 ₽` (пробел-разряды, запятая, символ после), проценты — 2 знака
  со знаком и истинным минусом `+12,34%` / `−4,10%`; тикеры/ISIN — uppercase mono.
  Каноничные форматтеры уже есть в `src/utils/format.ts`.
- Цветовая дисциплина: azure = только бренд; зелёный/красный = только P&L; violet =
  только ИИ-поверхности; amber = warning. Градиенты в хроме — нет (исключение —
  свечение ИИ-композера).
- Иконки — **Lucide** (`lucide-react`, ~2px, `currentColor`). Эмодзи — только как
  ведущий сигнальный глиф в проактивных сообщениях ИИ (📅 ⚖️ 🧾 ⚠️ 📌).
- Где предлагаются действия — дисклеймер «не индивидуальная инвестиционная рекомендация».

## Assets
